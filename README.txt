--------- Setup Instructions ---------
- Pre Requisites:
--- linux
--- docker installed and running
--- cuda toolkit installed
--- nvidia container toolkit installed


1. From a terminal prompt, clone the repo: git clone https://github.com/xXsurarusXx/CITYchat.git

2. enter into the repo: cd CITYchat

3. run setup.sh: ./setup.sh

4. initialize docker repo: docker compose -f docker_compose.yaml up
